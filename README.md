# Studio 14

Agencia creativa boutique — sitio web oficial.

## Servicios
1. Creación de contenido (fotografía, video, diseño gráfico)
2. Campañas Meta Ads (Instagram/Facebook)
3. Páginas web a medida

## Paleta
- `#FFF1B5` — cream yellow
- `#C1DBE8` — soft sky blue
- `#43302E` — dark brown

## Stack
- Astro 6
- Tailwind CSS v4
- Vercel adapter

## Comandos

```bash
npm install
npm run dev       # localhost:4321
npm run build
npm run preview
```
